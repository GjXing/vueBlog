## 使用方法

#### 下载项目到本地
```
git clone https://github.com/GJxing/vueBlog.git
```
#### 本地运行

> 1.安装依赖

```
npm install
```
安装了cnpm就用
```
cnpm install
```

> 2.在本地开启服务，然后就可以通过 http://localhost:8084 访问了

```
npm run dev


#### 发布到 GitHub Pages

> 1.打包

```
npm run build
